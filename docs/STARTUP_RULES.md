# 啟動守則 (每日讀取專案時請先閱讀)

## 目的
建立一致的工作流程，避免直接修改造成風險，並保存對話與備份紀錄。

## 規則
1. 不要直接修改檔案；需要修改時，先備份、再修改。
2. 對話紀錄都要備份。
3. 備份檔案必須支援繁體中文 (UTF-8 編碼)。
4. 每天執行讀取專案時，會先讀取 `README_FIRST.txt`。

## 每日開始流程 (建議)
1. 執行 `.\tools\readme-first.ps1`
2. 開始工作前若需修改檔案，先執行 `.\tools\backup.ps1`
3. 對話紀錄以 `.\tools\log-chat.ps1` 追加備份

## 備份位置
- 檔案/資料夾備份：`backups\YYYYMMDD\`
- 對話紀錄：`chat_logs\YYYY-MM-DD.txt`
